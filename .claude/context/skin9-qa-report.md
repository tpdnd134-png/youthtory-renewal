## P0 — 릴리즈 전 반드시 해결

### ~~P0-1. 로그인 페이지 — ID/PW 입력 폼 미표시~~ (해결)
- **상태**: 정상 — 로그인 상태에서는 member_login 모듈이 렌더링되지 않는 것이 카페24 기본 동작

### ~~P0-2. 네비게이션에 WOMEN 카테고리 없음~~ (의도적)
- **상태**: 정상 — 카페24 관리자에서 WOMEN 대분류를 의도적으로 미진열 상태로 설정

### P0-3. WOMEN 카테고리(cate_no=43) 제목이 "-Mens Essential"로 표시
- **심각도**: P1 (카테고리 구조 — 미진열 상태이므로 우선순위 하향)
- **증상**: `/product/list.html?cate_no=43` 접근 시 제목이 "-Mens Essential", 상품 2개만 표시
- **원인**: 카페24 관리자에서 카테고리명이 잘못 설정됨
- **수정**: 카페24 관리자 > 상품분류 > cate_no=43 이름을 "Womens SHOP"으로 변경 + 상품 배정

### ~~P0-4. 장바구니 체크박스 기본 미선택~~ (수정 완료)
- **상태**: 수정됨 — basket.html에 setInterval 기반 자동 전체선택 스크립트 추가
- **파일**: `order/basket.html` (2367행 부근)
- **스크린샷**: `qa/05-cart-mobile.png`

### P0-5. 상세페이지 이미지 카운터 "1 / 0"
- **심각도**: P0 (기능 오류)
- **증상**: 상품 상세페이지 이미지 슬라이더 하단에 "1 / 0" 표시 (총 이미지 수 0)
- **원인**: Swiper 또는 이미지 슬라이더 초기화 시 이미지 수 계산 실패
- **스크린샷**: `qa/04-detail-option-selected.png`

## UI/UX 개선 제안 (별도)

### 즉시 개선 가능
1. **New Arrivals 기본 탭을 Men으로**: Women 탭에 상품이 없으므로, 첫 로드 시 Men 탭 활성화
2. **장바구니 전체선택 기본 체크**: 카페24 기본 동작과 다른 사용자 혼란 유발
3. **할인 금액 표시 추가**: 장바구니 모바일 요약에서 할인 금액(-4,950원)이 별도 행으로 안 보임

### 디자인 개선
4. **상품 상세 — BUY NOW/CART 하단 고정바 위치**: 모바일에서 잘 작동하나, 옵션 미선택 시 안내 토스트 추가 권장
5. **빈 상태 디자인**: 리뷰 없는 경우, Q&A 없는 경우의 빈 상태 메시지 스타일링 필요
6. **폴더 배너 PC 레이아웃**: 1280px에서 폴더 이미지와 텍스트 겹침 없이 잘 표시되나, hover 인터랙션이 모바일 터치와 동일하게 작동 확인 필요

### 릴리즈 후 (P2)
7. **검색 오버레이 UI**: 현재 페이지 이동 방식, la-room 스타일 오버레이로 개선 가능
8. **페이지네이션**: 무한스크롤 또는 "더보기" 버튼으로 개선
9. **상품 카드 hover 효과**: PC에서 scale/opacity 트랜지션 누락
